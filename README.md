# nodechain

A blockchain-backed note taking application written in *rust*.

# Building on Manjaro

	sudo pacman -Sy rust
	cargo build

First build may be very long due to (300+) dependencies.

# Rust mode in emacs

At least in *emacs* 28 or if melpa is enabled :

	<M-x> package-install <RET>
	tust-mode <RET>
